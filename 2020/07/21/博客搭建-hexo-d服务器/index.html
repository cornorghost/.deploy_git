hexo+阿里云服务器搭建博客

配置的过程参考的是下面的链接：
https://blog.csdn.net/NoCortY/article/details/99631249
https://juejin.im/post/5d54f29af265da03cc08b45a

首先你得有个服务器，然后下面是我补充的遇到的问题

主机端安装hexo出现问题，可忽略

optional install error: Package require os(darwin) not compatible with your platform(win3）

没有可用软件包nginx

sudo yum install epel-release
yum update

图片不显示

官方文档插入图片时用这种方式：

{% asset_img test.jpg This is an test image %}

git上传文件没有同步到自定义的网页文件夹

  1. 可能是没有git钩子执行成功，尝试直接输入运行，看看文件夹里有没有文件，就是git-dir对应的地址
    git --work-tree=/home/www/website --git-dir=/home/git/blog.git checkout -f

2. 可能是自定义文件夹有访问权限，尝试在/home目录下使用 
    chmod -R 777 ./www
